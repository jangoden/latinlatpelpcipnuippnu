## üìñ Tentang Proyek

Project ini adalah platform web modern yang dibangun untuk memfasilitasi kegiatan kaderisasi tingkat lanjut (LATIN & LATPEL) oleh PC IPNU IPPNU Ciamis. Website ini berfungsi sebagai pusat informasi, pendaftaran peserta secara online, serta pengelolaan data peserta bagi panitia.

Dikembangkan dengan **Next.js App Router** terbaru untuk performa dan SEO yang optimal.

## ‚ú® Fitur Utama

* **Landing Page Interaktif:** Informasi lengkap mengenai TOR, Timeline, Fasilitas, dan Persyaratan dengan animasi yang halus.
* **Pendaftaran Online:** Form pendaftaran terintegrasi langsung dengan database.
* **Admin Dashboard:** Halaman khusus panitia untuk mengelola data pendaftar (`/admin`).
* **Dark Mode Support:** Mendukung tampilan gelap dan terang sesuai preferensi sistem pengguna.
* **SEO Optimized:** Menggunakan Metadata API Next.js, Sitemap, dan Robots.txt yang terkonfigurasi.
* **Responsive Design:** Tampilan optimal di Mobile, Tablet, dan Desktop.
* **Interactive UI:** Menggunakan komponen modern seperti Chatbot, Timeline, dan Glassmorphism effect.
